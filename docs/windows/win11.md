# Window 11

## 初期設定

### ローカルアカウント作成方法

1. 初期セットアップ中に「Shift」＋「F10」キーを押してコマンドプロンプトを開く。
2. コマンドプロンプトで「start ms-cxh:localonly」と入力する。
