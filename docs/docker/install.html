3.1. Docker Desktop インストール¶

https://www.docker.com/products/docker-desktop

3.1.1. インストール (Windows)¶

3.1.1.1. WSL2 のインストール¶

参考：https://docs.microsoft.com/ja-jp/windows/wsl/install-win10

3.1.1.1.1. Linux 用 Windows サブシステムを有効にする¶

管理者として PowerShell を開き、以下を実行します。

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

3.1.1.1.2. 仮想マシンの機能を有効にする¶

管理者として PowerShell を開き、以下を実行します。

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

Windows を再起動します。

3.1.1.1.3. Linux カーネル更新プログラムをパッケージをインストールする¶

3.1.1.1.3.1. ダウンロード¶

https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi

3.1.1.1.3.2. インストール¶

そのままインストーラを実行します。

3.1.1.1.4. WSL2 を既定のバージョンとして設定する¶

PowerShell を開いて、以下のコマンドを実行します。

wsl --set-default-version 2

3.1.1.2. インストーラのダウンロード¶

https://www.docker.com/products/docker-desktop

3.1.1.3. インストーラ起動¶

そのまま OK をクリックします。

3.1.1.4. インストール中¶

この処理は結構長いです。

3.1.1.5. インストール完了¶

Close and restart をクリックし、Windows を再起動します。
