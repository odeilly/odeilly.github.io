Pelicanで数式を書く

Date 2020年 7月年15日 (水) Tags Pelican / Math

Render Math Plugin の設定

Typogrify が必要となるので, pipコマンドでインストールします.

pip install typogrify

pelicanconf.py で, PLUGINS に render_math を追加します.

PLUGINS = [
             ...
             'render_math',
             ...
]

使用例

  • インライン数式

    インライン数式は, :math:`x^2 + x + 1 = 0` のように書きます.

    インライン数式は, \(x^2 + x + 1 = 0\) のように書きます.

  • 別行立て式

    .. math::

   S = \int_a^b f(x) dx.
    \begin{equation*} S = \int_a^b f(x) dx. \end{equation*}
