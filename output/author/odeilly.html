Pelicanを導入しました.

環境設定

ソフトウェアのインストール

ソフトウェア バージョン
Python 3.8.3
pelican 4.2.0
Markdown 3.1.1
ghp-import 0.5.5

  1. Pythonをインストールします. パスを通すようにして下さい.

  2. Pelicanをインストールします. Markdownで編集できるように, Markdownオプションを付けます.

    pip install pelican Markdown

  3. ghp-import をインストールします. GitHub Page のアップロードが楽になります.

    pip install ghp-import
